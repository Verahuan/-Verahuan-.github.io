子元素的onMouseDown和父元素的onMouseUp

Created2023-11-23|Updated2023-11-27
|Post Views:

问题描述

即使已经阻止了子元素的onClick事件的冒泡，onmousedown在子元素上，onmouseup在父元素上也会触发父元素的onClick事件

Discription

Even though you have prevented the onClick event of the child element from bubbling up, you still cannot prevent the parent element’s onClick handler from being notified when you handle onMouseDown on the child element and handle onMouseUp on the parent element.

解决方法

可以在父元素的点击事件中，通过判断当前文档中的activeElement是不是需要被阻止点击事件的子元素，如果是则不触发父元素的点击事件

Solution

You can determine whether to execute the onClick handler of the parent element by comparing the activeElement of the document with the child element that should be prevented from bubbling up. If the activeElement is the child element, you should choose to return and not execute the handler.
